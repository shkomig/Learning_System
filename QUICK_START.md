# 🚀 התחלה מהירה - מערכת למידת מתמטיקה

## המערכת כבר רצה! ✅

אם הגעת לכאן, שני השרתים אמורים לרוץ:
- ✅ **Backend**: http://localhost:3000
- ✅ **Frontend**: http://localhost:5173

## 📱 כיצד להשתמש במערכת

### 1. פתח דפדפן וגש ל: **http://localhost:5173**

### 2. דף הבית - Dashboard 
תראה:
- 📊 **4 כרטיסי סטטיסטיקות**:
  - נקודות ⚡
  - רמה 🏆
  - רצף ימים 🎯
  - תרגילים שהושלמו 📈

- 📈 **סרגל התקדמות** לרמה הבאה

- 📚 **7 נושאי לימוד**:
  - חיבור עד 10
  - חיבור עד 20
  - חיסור עד 10
  - חיסור עד 20
  - חיבור עד 100
  - חיסור עד 100
  - לוח הכפל

- 🏆 **10 הישגים** (badges) לפתיחה

### 3. התחל תרגול
1. לחץ על אחד מנושאי הלימוד
2. תרגיל חיבור יופיע עם ויזואליזציות צבעוניות
3. הקלד את התשובה
4. לחץ "בדיקה" או Enter
5. תקבל פידבק מיידי!

### 4. ניווט
בחלק העליון תמצא תפריט ניווט:
- 🏠 **בית** - Dashboard עם כל הסטטיסטיקות
- 📚 **תרגילים** - עוד תרגילים
- 🏆 **הישגים** - הישגים שפתחת
- 👤 **פרופיל** - המשתמש שלך
- ⚙️ **הגדרות** - התאמות

## 🎮 פיצ'רים מגניבים לנסות

### ✨ חגיגות על הצלחה
- כשתענה נכון - תראה אנימציה של sparkles ✨
- ניסיון ראשון נכון - בונוס של +5 נקודות! 🌟

### 💡 מערכת רמזים
- נתקעת? לחץ על כפתור "רמז" 💡
- אחרי 2 ניסיונות שגויים - רמז יופיע אוטומטית

### 🎯 מערכת נקודות
- **10 נקודות** על תשובה נכונה
- **+5 בונוס** בניסיון ראשון
- **+2 בונוס** על מהירות (פחות מ-10 שניות)

### 🏆 הישגים
הישגים שתוכל לפתוח:
1. 👣 **צעדים ראשונים** - תרגיל ראשון
2. 🔥 **עשר מושלם** - 10 תרגילים נכונים ברצף
3. ⚡ **מהיר כברק** - 10 תרגילים מתחת לזמן
4. ➕ **מלך החיבור** - 50 תרגילי חיבור
5. ➖ **מלך החיסור** - 50 תרגילי חיסור
6. 🏆 **לוחם השבוע** - 7 ימים רצופים
7. 💡 **עוזר חכם** - שימוש חכם ברמזים
8. 🗺️ **חוקר** - ניסית את כל הנושאים
9. 💯 **מועדון המאה** - 100 תרגילים
10. ⭐ **ציון מושלם** - 5 תרגילים בניסיון ראשון

### 📊 עליית רמות
- **רמה 1**: 0-100 נקודות
- **רמה 2**: 100-200 נקודות
- **רמה 3**: 200-300 נקודות
- וכן הלאה...

## 🛠️ בעיות?

### Frontend לא עובד?
```bash
cd frontend
npm run dev
```

### Backend לא עובד?
```bash
cd backend
npm run dev
```

### אתר לא נפתח?
ודא שהשרתים רצים ו-URL נכון: http://localhost:5173

### רוצה להתחיל מחדש?
כל הנתונים שלך נשמרים ב-localStorage בדפדפן.
רענן דף (F5) והכל ימשיך מאיפה שעצרת!
